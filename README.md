# safety-inspector
Vision app to detect risk behaviours and safety risks based on OSHA and NOM-036

Uses Mobile phone camara to detect in real-time safety risks based on a trained deep learning model.
Also can be used with a raspberry pi to continually monitor an area where safety must be enforced with the use of helmet, gloves, glasses and other equipment. Areas such as foundry, forging, and others with potential harm to humans.
